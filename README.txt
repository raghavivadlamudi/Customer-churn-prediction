# Customer Churn Prediction

This is a machine learning project to predict **customer churn** using the Telco dataset.

## Steps
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training (Logistic Regression, Random Forest, XGBoost)
- Model Evaluation (Confusion Matrix, ROC Curve)
- Best Model: Logistic Regression (Recall = 0.80)

## Dataset
[Telco Customer Churn Dataset - Kaggle](https://www.kaggle.com/blastchar/telco-customer-churn)
